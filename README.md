# HackMyVM Writeups

Welcome to my **HackMyVM** writeup repository.

This repository contains detailed walkthroughs of HackMyVM machines that I have completed as part of my cybersecurity and penetration testing practice. Each writeup documents my methodology from enumeration to privilege escalation, focusing on understanding the attack path rather than simply obtaining the flags.

## Objectives

- Practice real-world penetration testing techniques.
- Improve enumeration and privilege escalation skills.
- Document methodologies for future reference.
- Build a public portfolio showcasing practical cybersecurity experience.

## Writeup Structure

Each machine contains:

- Reconnaissance & Enumeration
- Vulnerability Discovery
- Initial Access
- Privilege Escalation
- Root Access
- Lessons Learned
- Supporting screenshots and commands
